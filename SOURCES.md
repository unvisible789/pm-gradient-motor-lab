# Sources

## ORNL 2010 Toyota Prius MG2 Benchmark

- T. A. Burress et al., *Evaluation of the 2010 Toyota Prius Hybrid Synergy
  Drive System*, ORNL/TM-2010/253, Oak Ridge National Laboratory, March 2011.
  PDF: https://info.ornl.gov/sites/publications/files/Pub26762.pdf

Values used in this repo:

- Table 4.1: 2010 Prius motor power rating, torque rating, speed rating, motor
  mass, motor volume, maximum dc-link voltage.
- Figure 3.11: 2010 Prius motor efficiency contours at 650 Vdc.

The ORNL report also contains locked-rotor torque versus position, back-EMF
testing, efficiency maps, and continuous-load duration tests. This repository
uses summary values until raw digitized curve data is added.
