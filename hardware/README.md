# Hardware - Motor Driver & Control System

This folder contains the circuitry, control logic, and build documentation for driving the PM-Gradient motor.

## Contents

- `circuit/`: Circuit schematics and descriptions
- `bom/`: Bill of Materials
- `firmware/`: Microcontroller code and logic
- `build/`: Step-by-step build and testing guides
- `safety/`: Safety recommendations and protection circuits

## Important Notes

- This is for **prototype / development** use.
- Start with a single EML channel before scaling up.
- The magnetic design is still being refined in FEMM. Hardware should be developed in parallel but not rushed.
- Always use current-limited power supplies during early testing.
