# Priority 2 EML/Stator Improvement Plan

Date: 2026-06-15

## Starting Point

The mesh-verified static geometry converges near zero/slightly negative net work. Treat earlier unrefined positive static results as artifacts until proven otherwise.

The best first-screen candidate is currently ASYM_B rotor geometry with +12 degree EML offset:

- Full-rev equivalent: 1.159 J/rev
- 45 degree work: 0.145 J
- Cancellation ratio: 0.934
- Status: hold for nearby variants, not mesh-qualified

## Design Hypothesis

Rotor-only asymmetry and trailing-edge rotor barriers did not break cancellation enough. The next leverage is stator/EML-side flux shaping:

- reduce EML pole width to lower backward pull,
- bias EML pole shoes toward assist windows,
- add trailing-side stator air relief,
- add back-iron shunts to alter flux return paths.

## Testing Order

1. Run the fine EML offset sweep first if not complete: +9, +12, +15, +18, +21.
2. If no offset exceeds the gate, screen `p2_eml_narrow_10/20/30`.
3. Screen `p2_eml_bias_mild`; run moderate only if mild improves cancellation.
4. Screen stator relief and shunt variants.
5. Run `p2_best_combo_mild` only if at least one individual feature improved the result.

## Promotion Gates

First-screen promotion requires either:

- at least 3 J/rev equivalent, or
- a large cancellation improvement compared with 0.934 plus stronger positive/negative torque imbalance.

Never promote an incomplete sweep. A 45 degree period screen must span essentially the full 45 degrees.

Mesh refinement gate:

- medium mesh,
- fine mesh,
- very-fine mesh if still positive.

No build or performance claims until the improvement survives mesh refinement.

## Energy Accounting

Use `validation/estimate_pulse_energy.py` before discussing input/output power for energized EML assist. The default report is only a placeholder budget; replace with measured coil inductance, resistance, current waveform, driver losses, and recovery efficiency.

Static PM work and energized EML work must not be mixed. The EML pulse input must be subtracted from any added mechanical work.
